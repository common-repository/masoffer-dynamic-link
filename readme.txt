== Description ==

This plugin converts all links to products in the article into Affiliate Tracking links of MasOffer if publisher qualifies.
Please see the Offer available to you at https://pub.masoffer.com/offer.

With private offers, sign up for Offer to make sure Affiliate Tracking works properly.

== Installation ==

After installing the plugin from WordPress you need to have an account at the website https://pub.masoffer.com/.
Publisher ID is the username when you register your account at https://pub.masoffer.com/.
Please enter and save this Publisher ID at the information settings section of the Masoffer Dynamic Link Plugin:
Admin menu -> Settings -> Masoffer Dynamic Link
URL Path: Yourweb.com/wp-admin/options-general.php?page=masoffer-dynamic-link-official
